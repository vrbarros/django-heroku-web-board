## Quick Start with virtualenv

1. Create a virtualenv with Python 3.8

```bash
virtualenv venv -p python3.8
```

2. Activate the virtualenv

```bash
source venv/bin/activate
```

3. Install requirements.txt

```bash
pip install -r requirements.txt
```
